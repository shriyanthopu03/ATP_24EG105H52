# Week 4

This folder contains two HTML and CSS practice assignments.

## Contents

- `Ass1.html` - basic HTML structure practice with headings, paragraphs, and lists
- `Ass2.html` - cat profile layout with linked stylesheet
- `Ass2.css` - styling for the second assignment

## Assignment Details

### Assignment 1

Practice page using plain HTML elements such as:

- headings
- paragraphs
- unordered lists

### Assignment 2

A simple profile-style page that includes:

- a heading and intro section
- an image
- profile details in an ordered list
- external links
- a small CSS file to float the image

## Notes

- These files are static front-end exercises.
- Open the HTML files directly in a browser to view the results.
