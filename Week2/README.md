# Week 2

This folder contains JavaScript practice problems and small mini-projects focused on core language concepts.

## Contents

- `1.js`, `2.js` - basic JavaScript exercises
- `Array-operations-advanced/` - array manipulation practice
- `Library Book Management System.js` - simple book management logic
- `Shallow & Deepcopy/` - copy behavior examples
- `Spread operator & Rest pa/` - spread/rest operator practice
- `Task Management System/` - small task manager app
- `Timer functions/` - timer function examples
- `backend1/` - Express backend with user and product routes

## Backend 1

The `backend1` folder contains a basic Express app with separate routers for users and products.

### Run

```bash
cd Week2/backend1
npm install
node server.js
```

### Routes

- `/user-api/users`
- `/product-api/products`

## Notes

- This week focuses on core JavaScript fundamentals and REST API basics.
- The backend stores data in memory, so values reset when the server restarts.
